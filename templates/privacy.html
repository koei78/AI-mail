プライバシーポリシー

最終更新日：2026年4月2日

「MailHub」（以下「本サービス」）は、「運営者名」（以下「当社」）が提供するメール管理・自動分類サービスです。当社は、お客様の個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条　事業者情報

項目内容
事業者名「会社名または氏名」
住所「住所」
メールアドレス「連絡先メールアドレス」

第2条　取得する個人情報

当社は、以下の個人情報を取得します。

（1）アカウント情報

  • 氏名・メールアドレス（Google / Microsoft アカウントから取得）
  • プロフィール画像（任意）

（2）メール接続情報

  • 独自ドメインメール用のIMAPホスト・SMTPホスト・ポート番号・ユーザー名
  • メールパスワード（Fernet方式で暗号化してサーバー側DBに保存します）
  • Google / Microsoft OAuth2リフレッシュトークン（Fernet方式で暗号化してサーバー側DBに保存します）

（3）メール情報

本サービスの機能提供のため、以下のメールデータにアクセスします。

  • メールの件名・本文・送受信日時・送受信者のメールアドレス
  • メールのラベル・フォルダ情報
⚠️ 添付ファイルの内容は取得しません。

（4）AI分類結果

AIによるメール分類処理の結果として、以下の情報を当社サーバー側のデータベースに保存します。

  • メールの件名・送信者・AI生成の要約文・優先度カテゴリ（A/B/C）

メール本文はIMAPサーバーから直接取得し、AI分類処理後に当社DBへは保存しません（AI分類結果のみ保存）。

（5）バックグラウンド自動同期

本サービスは、お客様のご利用状況に関わらず、15分おきに自動でメールアカウントの同期処理を実行します。これはメール一覧を最新の状態に保つためのものです。

（6）利用情報

  • IPアドレス・ブラウザ情報・デバイス情報
  • 本サービスの操作ログ・アクセス履歴
  • Cookie および類似技術による収集した情報

（7）決済情報

クレジットカード番号等の決済情報は、当社では保持しません。決済代行サービス（Stripe等）が直接取り扱います。

第3条　個人情報の利用目的

取得した個人情報は、以下の目的に利用します。

  1. 本サービスの提供・運営・改善
  2. メールの自動分類・優先度付け機能の実行
  3. ユーザーへのサポート・お問い合わせ対応
  4. 利用規約違反・不正利用の調査・対応
  5. 料金の請求・決済処理
  6. サービスに関する重要なお知らせの送信
  7. 法令に基づく対応

第4条　AI機能とデータの外部送信

（1）AI処理の概要

本サービスはメールの自動分類・優先度付けに人工知能（AI）を使用します。この処理のため、お客様のメール情報の一部を外部AIサービスに送信します。

（2）送信先・送信内容

項目内容
送信先サービスOpenAI, Inc.（モデル: gpt-4o-mini）
送信されるデータメールの件名・本文（一部または全文）
送信しないデータ添付ファイル・パスワード・決済情報
利用目的メール分類・優先度判定のみ
モデル学習への利用なし（OpenAIのAPIポリシーに基づき、学習には使用されません）
AIサービス側での保持期間処理完了後に破棄（同社のデータ保持ポリシーに準拠）

（3）ユーザーの選択肢

アカウント設定からAI分類機能を無効にすることができます。無効にした場合、メールの自動分類機能はご利用いただけません。

（4）各社のプライバシーポリシー

  • OpenAI: https://openai.com/policies/privacy-policy

第5条　第三者提供

当社は、以下の場合を除き、お客様の個人情報を第三者に提供しません。

  1. お客様の同意がある場合
  2. 法令に基づく場合（裁判所・警察等からの適法な要請）
  3. 生命・身体・財産の保護のために必要な場合
  4. 業務委託先への提供（第6条参照）

第6条　業務委託先（サービス提供に利用する第三者サービス）

当社は、以下のサービスを利用してサービスを提供します。各社の個人情報の取り扱いは、各社のプライバシーポリシーに従います。

サービス用途提供者
Google OAuth 2.0Gmail 連携認証Google LLC
Microsoft OAuth 2.0Outlook 連携認証Microsoft Corporation
OpenAI APIAI メール分類処理OpenAI, Inc.
Stripe決済処理Stripe, Inc.
クラウドサーバー等データホスティング（例: Amazon Web Services, Inc.）

第7条　Cookieの使用

当社は、以下の目的でCookieおよび類似技術を使用します。

  • ログイン状態の維持
  • セキュリティトークンの管理
  • 利用状況の分析（Google Analytics等）

ブラウザの設定によりCookieを無効にすることができますが、その場合、本サービスの一部機能が利用できなくなります。

第8条　データの保管・セキュリティ

保管期間

  • アカウント情報：アカウント削除後90日間
  • AI分類結果（件名・送信者・要約）：アカウント削除後90日間
  • メールデータのキャッシュ：「7日間」（設定により変更可能）
  • 利用ログ：「1年間」

セキュリティ対策

  • 通信の暗号化（TLS/SSL）
  • パスワード・OAuth2トークンのFernet方式暗号化
  • データベースの暗号化
  • アクセス権限の最小化
  • 定期的なセキュリティレビュー

第9条　お客様の権利

お客様は、当社に対して以下を請求することができます。

  • 個人情報の開示・訂正・削除
  • 利用停止・消去
  • 第三者提供の停止

請求は、本ポリシー末尾のお問い合わせ窓口までご連絡ください。本人確認のうえ、合理的な期間内に対応します。

第10条　未成年者の利用

本サービスは、16歳未満の方を対象としていません。16歳未満の方の個人情報を意図せず取得した場合、速やかに削除します。

第11条　Google APIの利用について

本サービスによるGoogleユーザーデータの取得・使用は、Google API サービスのユーザーデータポリシー（Limited Use 要件を含む）に準拠しています。

取得したGmailデータは以下の目的にのみ使用します。

  • お客様へのメール分類機能の提供
  • お客様が明示的に要求した機能の実行

Gmailデータを以下の目的に使用することはありません。

  • 広告の表示・ターゲティング
  • データの販売・第三者提供（本ポリシーに定める場合を除く）
  • 人が読む形でのデータ閲覧（セキュリティ・法令対応・サポートを除く）

第12条　プライバシーポリシーの変更

当社は、法令の改正やサービス内容の変更に伴い、本ポリシーを変更することがあります。重要な変更がある場合は、サービス上またはメールにてお知らせします。変更後に本サービスを継続利用した場合、変更後のポリシーに同意したものとみなします。

第13条　お問い合わせ

個人情報の取り扱いに関するお問い合わせは、以下までご連絡ください。

メールアドレス：「連絡先メールアドレス」
